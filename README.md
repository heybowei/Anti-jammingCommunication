This is used for Anti-jamming Communication for both analog beamforming and digital beamforming.

gr-mmwave control is for digital beamforming;
  -Install GNU Radio first. https://wiki.gnuradio.org/index.php/InstallingGR
  -Rebuild this project on your machine.
  -Run the project with USRP.

pycontrol-mmwave is for analog beamforming:
  -Run with GUI: ./run.sh
  -Run with script: sudo python3 beamlisten.py
